## Тестирование

В проекте используется фреймворк `pytest` для проведения автоматизированного тестирования.

### Запуск тестов:
Для запуска всех тестов выполните команду:
`pytest`

### Отчет о покрытии (Coverage):
Для генерации HTML-отчета о покрытии тестами используйте:
`pytest --cov=src --cov-report=html`

После выполнения команды отчет будет доступен в папке `htmlcov/`.
