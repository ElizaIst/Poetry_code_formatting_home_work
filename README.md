# Виджет банковских операций

Проект предназначен для обработки и анализа данных о банковских операциях клиентов. Он предоставляет инструменты для фильтрации транзакций по их статусу и сортировки по дате.

## Основные функции

- **Фильтрация по статусу (`filter_by_state`):** Позволяет получить список операций с конкретным состоянием (например, только исполненные — `EXECUTED`).
- **Сортировка по дате (`sort_by_date`):** Позволяет упорядочить транзакции от самых свежих к старым или наоборот.

## Установка и использование

1. Клонируйте репозиторий.
2. Убедитесь, что у вас установлен Python 3.10+.
3. Функции можно импортировать из модуля `src/processing.py`.

## Инструменты качества кода
В проекте используются:
- **flake8** для проверки стиля кода (PEP 8).
- **mypy** для проверки аннотаций типов.
