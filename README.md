# Проект: Банковские операции

## Описание
Этот проект содержит инструменты для обработки банковских транзакций, маскировки номеров карт/счетов и логирования.

## Новый функционал: Декораторы
В модуле `src/decorators.py` реализован декоратор `log`, который:
* Автоматически регистрирует начало и конец выполнения функций.
* Логирует результаты успешного выполнения.
* В случае ошибки записывает её тип и входные параметры функции.
* Поддерживает вывод логов как в консоль, так и в указанный файл через аргумент `filename`.
